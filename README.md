# Odin Recipes

### Description
This project is part of The Odin Project curriculum. It is a basic recipe website where I will practice and demonstrate my foundational HTML skills.

### Demonstrated Skills
By the end of this project, I will have demonstrated the following:
* **HTML Structure**: Creating well-organized documents using `html`, `head`, and `body` tags.
* **Text Formatting**: Using headings, paragraphs, and lists (ordered and unordered).
* **Media & Links**: Implementing images with proper paths and connecting pages using anchors.
* **Git & GitHub**: Managing version control, commits, and pushing code to a remote repository.
* **Linux Terminal**: Navigating directories and managing files using the command line.
